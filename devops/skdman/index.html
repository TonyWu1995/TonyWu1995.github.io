Sdkman

2021-09-21 September Tuesday · 1 min · Tony

介紹#

  • SDKMAN是一款基於Linux/Mac的SDK管理工具，可以管理多個版本JDK，也可以安裝與JDK相容的套件

安裝SDKMAN#

$ curl -s "https://get.sdkman.io" | bash
$ source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk version
> sdkman 5.0.0+51

羅列Java版本#

sdk list java

安裝JAVA#

sdk install java x.y.z-open

移除#

sdk uninstall java ...

指定預設JAVA#

sdk default java 11
