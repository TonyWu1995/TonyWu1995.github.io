Mongo備份與還原

介紹 在工作上不免要定期備份database的資料與還原datdabase的資料 備份步驟 連線進入mongo mongo -u <username> -p <password> 切換db use admin 磁碟同步與上鎖 db.runCommand({fsync:1,lock:1}) 確認是否上鎖 db.currentOp() 備份資料 mongodump --host <host-ip> --port 27017 --db <database> --authenticationDatabase admin --username <username> --password <password> --out <Path> 解鎖 db.fsyncUnlock() 還原步驟 還原 mongorestore --host <host-ip> --port 27017 --username <username> --password <password> --db <database> <backup file path> 參考連結 Mongo
